### Identifiers
- STIG ID: WN11-CC-000210
- Vuln ID: V-253269 (DISA STIG)

## System / Scope

- **System:** Microsoft Windows 11 Pro 64-bit (Azure virtual machine, x64), standalone
- **Environment:** Azure-hosted virtual machine used for STIG testing
- **Role of system:** User workstation endpoint used for security testing and STIG remediation within a controlled lab environment

## Finding

- Windows Defender SmartScreen was not fully enforced.
- Users could bypass protection warnings.

## Security Impact

- Increased risk of executing malicious applications.
- Reduced endpoint protection effectiveness.

## Remediation Performed

- Reviewed SmartScreen configuration.
- Enabled SmartScreen and enforced block mode.

## PowerShell Remediation Script

```powershell
New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\System' -Force | Out-Null

Set-ItemProperty `
  -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\System' `
  -Name 'EnableSmartScreen' `
  -Value 1 `
  -Type DWord

Set-ItemProperty `
  -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\System' `
  -Name 'ShellSmartScreenLevel' `
  -Value 'Block'

```
Validation

Confirmed the registry values were correctly set.

Performed a Tenable compliance rescan using the Microsoft_Windows_11_STIG policy.

Verified that STIG WN11-CC-000210 passed successfully after remediation.
