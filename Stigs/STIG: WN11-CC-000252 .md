# STIG: WN11-CC-000252 â€“ Windows Game Recording and Broadcasting must be disabled

### Identifiers
- STIG ID: WN11-CC-000252
- Vuln ID: V-253278 (DISA STIG)

## System / Scope

- **System:** Microsoft Windows 11 Pro 64-bit (Azure virtual machine, x64), standalone
- **Environment:** Azure-hosted virtual machine used for STIG testing
- **Role of system:** User workstation endpoint used for security testing and STIG remediation within a controlled lab environment

## Finding

- Windows Game Recording and Broadcasting features were enabled.
- Background recording services were active.

## Security Impact

- Potential exposure of sensitive on-screen information.
- Unnecessary system resource usage.

## Remediation Performed

- Reviewed Game DVR configuration.
- Disabled Windows Game Recording and Broadcasting via registry policy.

## PowerShell Remediation Script

```powershell
New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\GameDVR' -Force | Out-Null

Set-ItemProperty `
  -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\GameDVR' `
  -Name 'AllowGameDVR' `
  -Value 0 `
  -Type DWord

```
Validation

Confirmed the registry value was correctly set.

Performed a Tenable compliance rescan using the Microsoft_Windows_11_STIG policy.

Verified that STIG WN11-CC-000252 passed successfully after remediation.
